# 重新出发

走过岁月我才发现世界多不完美，成功或失败都有一些错觉，沧海有多广江湖有多深，局中人才了解……

时间过真的很快，不知不觉，自己已经走进了29岁的生活，回首走过的这些年，这些月、这些日子，发现靠记忆去记录的生活，过去的很多光影都已经搁浅在了记忆里，任凭自己怎么努力的回想，能找回的、想起的总是只有零星点点，无论是生活还是工作。

或许是因为进入2022年后工作特别的繁忙，过的特别的累，所以最近的焦虑的情绪越来越严重， 于是乎在心里告诉自己，这样下去一定会出问题，一定无论如何也要在这次项目交付以后，好好的整理一下生活，复盘一下自己！

今天，正好恰逢周五、也正好是2022年下半场的第一天，有个相对空闲的时间让自己梳理、复盘一下自己，调整调整自己的状态，重新出发，于是，决定从这篇博客开始，让写博客这件事成为自己能够长期坚持的一件事，更是为了让自己能够在匆匆忙忙的日子中留下自己的一些工作上、生活上所想所感所悟……

这一次，重新出发，太初有为、做时间里的旅人！！！
