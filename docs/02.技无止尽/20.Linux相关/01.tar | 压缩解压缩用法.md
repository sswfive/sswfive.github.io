# 文件压缩&解压


linux 默认支持的压缩格式：
```
.gz
.bz2
.zip
```
- .gz和.bz2的压缩包需要使用tar命令来压缩和解压缩
- .zip的压缩包需要使用zip来压缩，使用unzip命令来解压缩

### tar命令的使用

	计算机中的数据经常需要备份，tar是Unix/Linux中常用的备份工具，此命令可以把一系列文件归档到一个大文件中，也可以把档案文件解开恢复数据。

tar使用格式：

```shell
多文件归档:	   tar [选项]	打包文件名	(文件1	文件2)  
目录归档:	    tar [选项] 打包文件名	 目录
```

常用选项：

| 选项 | 含义                                                       |
| ---- | ---------------------------------------------------------- |
| -c   | 生成归档文件，创建出来了一个打包文件                       |
| -v   | 显示打包或者解包的详细过程                                 |
| -f   | 指定文件名为`.tar`这个格式的文件，必须放在所有选项的最后面 |
| -x   | 解档文件，将数据从打包文件中恢复出来                       |


tar命令演示：

- 打包一个文件

  - `tar -cvf 打包文件名.tar 要打包的内容`

  - 示例：

    `tar -cvf test.tar code`

- 解包一个文件

  - `tar -xvf 打包的文件名`

  - 示例：

    `tar -xvf test.tar`

> 注意点：
>
> 1. tar 命令在使用选项的时候，在选项前可以使用`-`，不使用`-`。
>
> 2. 除了f 需要放在选项的最后，其他的选项位置顺序随意。


# 归档压缩&解压解包

tar命令和zip命令联合使用实现文件的 打包压缩 和 解压解包

归档压缩

	格式：
	
		`tar -zcvf 压缩包文件名.tar.gz 待压缩的文件或目录`
	
	示例：
	
	 tar -zcvf test.tar.gz 1.txt   将`1.txt`文件压缩为`test.tar.gz`文件



解压解包

	格式：
	
		`tar -zxcf 压缩包文件名.tar.gz`
	
	示例：
	
		`tar -zxcf text.tar.gz`



解压解包 到制定的目录中

	格式：
	
		`tar -zxcf 压缩包文件名.tar.gz -C 指定目录` 
	
	示例：
	
		`tar -zxcf text.tar.gz -C /home/python/Destop/code`

```
选项	说明
-c	创建打包文件
-v	显示打包或者解包的详细信息
-f	指定文件名称, 必须放到所有选项后面
-z	压缩或解压缩(.gz)
-j	压缩或解压缩(.bz2)
-x	解包
-C	解压缩到指定目录

```
### zip和unzip命令及选项的使用
```
命令	说明
zip	    压缩成.zip格式文件
unzip	解压缩.zip格式文件

unzip命令选项:

选项	说明
-d	    解压缩到指定目录
```


压缩文件：

	格式：
	
		zip [-r] 压缩文件 源文件
	
	示例：

​	       	`zip -r a.zip a`  因为目标文件也可以不指定扩展名，所以也可以这么写`zip -r a a`

解压文件：

	格式：
	
		unzip 压缩文件
	
	示例：
	
		unzip a.zip  将在a.zip这个文件解压到当前目录下

>注意点：
>
>通过zip压缩文件的目标文件不需要指定扩展名，默认扩展名为zip。
