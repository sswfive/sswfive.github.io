## 切片操作

- 通过索引区间访问线性结构的一段数据
- sequence[start:stop] 表示返回[start,stop)区间的子序列
- 支持负索引
- start为0可以省略
- stop为0可以省略
- 超过上界，就取到末尾，超过下界，取到开头
- start一定要在stop的左边。
- [:]表示从头至尾，全部元素被取出，等效于copy()方法
