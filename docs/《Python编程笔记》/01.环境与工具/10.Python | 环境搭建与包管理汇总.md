## Python安装方式：
1. 源码安装（适用于：Linux、MacOS）
2. 图形界面安装（适用于：MacOS、Win）
3. Conda安装（适用于：Linux、MacOS、Win）
### 安装说明
> 安装的是python3.8版本
> [安装包官网下载地址](https://www.python.org/downloads/release/python-3813/)

#### 方案一：源码安装
```bash
# 下载
wget https://www.python.org/ftp/python/3.8.13/Python-3.8.13.tgz

# 解压
tar -zxvf Python-3.8.13.tgz 

# 进入解压后的目录
cd Python-3.8.13

# 安装编译
./configure --prefix=/usr/local/python3.8
make
make install

# 验证
python -V

# 设置软连接
ln -s /usr/local/python3.8/bin/python3.8 /usr/bin/python3
ln -s /usr/local/python3.8/bin/pip3.8 /usr/bin/pip3
```

## 环境管理选型方案

1. [venv](https://docs.python.org/zh-cn/3/library/venv.html#module-venv): Pthon标准库中的虚拟环境管理工具
2. [conda](https://docs.conda.io/en/latest/): Anaconda 下的环境与包管理工具
3. [virtualenv](https://virtualenv.pypa.io/en/latest/): 第三方的虚拟环境管理工具
4. [pipenv](https://pipenv.pypa.io/en/latest/): 第三方的虚拟环境管理工具
5. [poetry](https://python-poetry.org/): 是一个依赖管理和打包工具
6. [pdm](https://github.com/pdm-project/pdm): 一个现代的 Python 包管理器，支持 PEP 最新标准
