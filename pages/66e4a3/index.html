Python每日一库总览

# 文件处理

  • pathlib: 一款使用面向对象的方式操作路径的内置库

# 数据处理

  • PyFlink: 一款基于Python的Flink库,用于分布式实时计算
  • Numpy: 一个在Python中做科学计算的基础库
  • Pandas: 一款基于Numpy构建的分析结构化数据的工具集

# 进程管理

  • subprocess: 一款可以在Python程序内执行外部命令的内置库
  • supervisord: 一款用于Linux操作系统的进程管理工具

# 任务处理

  • APScheduler: 一款Python开发的定时任务工具。

# ModelOps

  • MLFlow: 一款用于机器学习模型的全生命周期管理的库
