VIM常用命令总结

# 编辑(插入)模式

# 插入

  • i: insert ; I: insert before line
  • a: append A: append after line
  • o: open a line below O: append a line above

# 删除

ctrl + h   # 删除上一个字符
ctrl + w   # 删除上个单词
ctrl + u   # 删除当前行

# 单词间移动

b/B   # 回到上一个word/WORD开头，可以理解为backword

# word指的是以非空白符分割的单词，
# WORD指的是以空白分割的单词
